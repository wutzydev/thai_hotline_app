# 📱 Thai Hotline App (สายด่วน THAILAND)

แอปพลิเคชันรวบรวมเบอร์โทรศัพท์สายด่วนที่สำคัญในประเทศไทย พัฒนาด้วย **Flutter** ออกแบบ UI ให้สวยงาม ใช้งานง่าย และแบ่งหมวดหมู่ชัดเจน พร้อมฟีเจอร์กดโทรออกได้ทันทีเพื่อความสะดวกรวดเร็วในยามฉุกเฉิน

## ✨ ฟีเจอร์หลัก (Features)
- **Splash Screen:** หน้าต้อนรับเข้าสู่แอปพลิเคชันหน่วงเวลา 3 วินาที
- **หมวดหมู่สายด่วน 4 หมวดหลัก:**
  - 🚗 การเดินทาง
  - 🚑 อุบัติเหตุ-เหตุฉุกเฉิน
  - 🏦 ธนาคาร
  - 💧 สาธารณูปโภค
- **One-Tap Calling:** เชื่อมต่อระบบโทรศัพท์ของตัวเครื่อง สามารถกดปุ่มเพื่อโทรออกเบอร์สายด่วนได้ทันที (ใช้งาน `url_launcher`)
- **About UI:** หน้าแสดงข้อมูลส่วนตัวของผู้จัดทำ
